# Библиотека анимационных блоков

Проект устроен как каталог отдельных демо-страниц.

- `/` - главная страница со списком блоков.
- `/orion/` - блок созвездия эксперта из `/Users/sergej/Web09.07`.

Структура для новых блоков:

```text
new-block/
  index.html
src/blocks/new-block/
  styles.css
  main.js
```

После добавления нового блока достаточно добавить объект в `src/library.js`.
